Source:  https://cloud.google.com/speech-to-text/docs/quickstart-client-libraries

1) Set the environment variable GOOGLE_APPLICATION_CREDENTIALS to the file path of the JSON file that contains your service account key.
set GOOGLE_APPLICATION_CREDENTIALS=[PATH]

2) Download the Cloud SDK installer.
3) Launch the installer and follow the prompts (install Bundled Python).
4) After installation has completed, accept the following options:
	Start Cloud SDK Shell
	Run gcloud init
